# Proposta Solar

Aplicação web para ler contas de energia, dimensionar sistemas fotovoltaicos e gerar propostas comerciais profissionais em PDF.

## Uso local

Execute `ABRIR-APP.bat` ou rode `npm start` e acesse o endereço exibido no terminal.

## Desenvolvimento

- `npm test`: executa os testes automatizados.
- `npm run build`: gera o pacote estático de produção em `dist/`.
